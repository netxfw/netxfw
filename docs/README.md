# netxfw Documentation

Welcome to the netxfw documentation. This directory contains all the documentation for the netxfw project organized by category.

## Directory Structure

- [standalone/](./standalone/) - Documentation for standalone version of netxfw
- [cluster/](./cluster/) - Documentation for cluster version of netxfw (coming soon)
- [api/](./api/) - API reference documentation
- [cli/](./cli/) - Command-line interface documentation
- [plugins/](./plugins/) - Plugin development guides
- [testing/](./testing/) - Testing documentation and procedures
- [performance/](./performance/) - Performance benchmarks and optimization guides
- [cloud/](./cloud/) - Cloud environment support documentation
